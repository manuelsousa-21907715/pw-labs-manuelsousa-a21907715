CSS

CSS serve para adicionar estilos á pagina.

Seletores de CSS
Elementos Usado Descrição
Universal done O seletor universal (*) aplica o estilo a todos os elementos da página.
Tipo done O seletor de tipo usa o nome do marcador a estilizar.
Classe done O seletor de classe associa um nome a um estilo.
Id done O atributo id permite identificar univocamente um elemento do HTML
Atributo done Seletores que se aplicam a atributos de elementos.
Pseudo-Class done Seletor associado ao estado de um elemento.
Pseudo-Elemento done Seletor associado em partes de um elemento.
